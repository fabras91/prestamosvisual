Microsoft Visual FoxPro 9.0 Prerequisites

Components with Language Version Conflicts

Visual FoxPro setup requires that certain system components be installed in the same language version as the language version of your operating system. However, Visual FoxPro setup does not include the language version that your operating system requires for these components.

You need to download the correct language version from the Internet, using the links in "Required matching language versions" later in this readme. When you have installed all components required for your operating system, you can return to setup by running Setup from the Visual FoxPro CD.

Also, you can install matching language versions for additional system components using the links "Optional matching language versions" later in this readme. These components are not required to be in the same language version as the language version of your operating system. If you do not choose to install the matching language version, these components will be installed in the default language version, English, when you return to Visual FoxPro setup.

1.1 Required matching language versions

Locate your operating system below and install the correct language version for the components listed.

Note: It is recommended that you print or bookmark this page prior to installing the components listed below. This page does not automatically display after a machine re-start.

Windows XP

Windows XP Service Packs
For information about downloading and installing this component, see http://www.microsoft.com/windowsxp/default.asp.

 

Windows 2000

Windows 2000 Service Pack 3 or higher
For information about downloading and installing this component, see http://www.microsoft.com/windows2000/downloads/servicepacks/default.asp.

1.2 Optional matching language versions available

Setup does not include a language version that matches the language version of your operating system for the following components. You can, however, proceed with Visual FoxPro setup and install the English language version. If you would like to install a matching language version, use the links below.

Microsoft Internet Explorer 6.0 and Internet Tools

For information about downloading and installing this component, see http://www.microsoft.com/windows/ie/. Hebrew and Arabic language versions require a setup specific to their respective languages.

For other languages, you may choose to install the English version, which is compatible. To do so, run ie6setup.exe from the IE60 folder on the Visual FoxPro CD. However, Internet Explorer will contain English in the user interface. If you would like to avoid this, please see http://www.microsoft.com/Windows/IE/intl/default.asp for more information about downloading Internet Explorer in the language of your operating system.

Return to Setup

When you have installed all the components required for your operating system, you can return to setup by running Setup from your Visual FoxPro CD.
