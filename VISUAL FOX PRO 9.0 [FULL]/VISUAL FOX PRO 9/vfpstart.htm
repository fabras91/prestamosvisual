Microsoft Visual FoxPro 9.0 Setup

 

Welcome to Visual FoxPro 9.0!

This page provides information about installing Visual FoxPro 9.0.
For additional setup information and late-breaking product information, consult the Visual FoxPro Readme.

This version of Visual FoxPro requires an update to the Windows components on your computer. Depending on the current state of your computer, this may involve restarting your computer a few times. When the component update is finished, you can continue installing the Visual FoxPro components.

To install Visual FoxPro

  1. Insert the Visual FoxPro CD and from your desktop, click My Computer to open the explorer.
  2. In the explorer, open the node for the CD-ROM drive and locate Setup.exe at the root level of the CD.
  3. Double-click Setup.exe to launch the Visual FoxPro Installation wizard.
  4. Click Prerequisites to update the required system components.
  5. Accept the End User License Agreement and click Continue.
  6. To have Setup automatically log on to your computer after restarts, select the Automatically log on option and enter the appropriate information. Then click Continue.
  7. Click Install Now! to continue with setup. This may require your computer to restart more than once depending on the updates installed. When the update is complete, click Done to return to the Installation wizard.
  8. On the Visual FoxPro Setup screen, click Visual FoxPro to install the Visual FoxPro 9.0 components.
  9. On the Start page of the Installation wizard, accept the EULA, verify your name, and click Continue.
  10. On the Options page of the Installation wizard, choose the features you want to install and click Install Now! to begin installation.

After you have updated the system software components, you can open VFPStart.hta at the root level of the Visual FoxPro CD for information about installing additional components included with Visual FoxPro, such as InstallShield Express and the Microsoft SQL Server Desktop Engine.
